ScatterBrain Documentation
==========================

ScatterBrain is a Python library for loading, processing, analyzing, modeling,
and visualizing Small-Angle X-ray Scattering (SAXS) and Wide-Angle X-ray
Scattering (WAXS) data.

.. note::

   This project is under active development (Pre-Alpha).
   Features and the public API are subject to change.
